# Agents Context

This folder contains context files for AI coding agents working on the ai-secretary project.

## Files

### CURRENT_TASK.md
- Project overview and status
- Current sprint goals
- What's completed vs in progress
- Next steps in priority order
- Known issues and blockers

### DECISIONS.md
- Log of architectural decisions
- Rationale for each choice
- Alternatives considered
- Backlog decisions (not yet made)

### RESEARCH/
- Collected research on various topics
- See `RESEARCH/index.md` for overview

## Usage

When starting a new session:
1. Read `CURRENT_TASK.md` to understand current state
2. Read `DECISIONS.md` to understand past decisions
3. Check `RESEARCH/` for relevant topics

When making decisions:
1. Document the decision in `DECISIONS.md`
2. Update `CURRENT_TASK.md` with progress
3. Add research to `RESEARCH/` if needed

## Guidelines

- Keep files concise and actionable
- Update status frequently
- Document why decisions were made, not just what
- Note alternatives that were rejected
- Flag decisions that need review
